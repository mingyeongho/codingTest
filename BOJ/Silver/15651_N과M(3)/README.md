### Algorithm: 백트래킹

> [!abstract] 요구사항 분석
> 시간 제한: 1초
> 메모리 제한: 512MB

> [!question] 문제
> 자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.
>
> - 1부터 N까지 자연수 중에서 M개를 고른 수열
> - 같은 수를 여러 번 골라도 된다.

> [!info] 입력
> 첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)

> [!success] 출력
> 한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.
>
> 수열은 사전 순으로 증가하는 순서로 출력해야 한다.

> [!example] 예시1
> input
>
> ```
> 3 1
> ```
>
> output
>
> ```
> 1
> 2
> 3
> ```

> [!example] 예시2
> input
>
> ```
> 4 2
> ```
>
> output
>
> ```
> 1 1
> 1 2
> 1 3
> 1 4
> 2 1
> 2 2
> 2 3
> 2 4
> 3 1
> 3 2
> 3 3
> 3 4
> 4 1
> 4 2
> 4 3
> 4 4
> ```

> [!abstract] 내가 이해한 내용
> N과M(1)과 비슷한데 중복된 수가 올 수 있음.

> [!tip] 문제의 본질 (어떠한 것을 나에게 물어보고 있는지)
> 백트래킹을 사용해서 구현할 수 있냐

> [!hint] 아이디어
> 기존에는 isUsed 배열이 존재했는데 여기서는 필요없음.
> n중 for 문을 돌린다는 느낌으로 구현
