> [!question] Question
> 모든 글자가 A, B로 바뀜
> 단어 위로 아치형 곡선을 그어 같은 글자끼리 쌍을 짓기로 하였다.
> 선끼리 교차하지 않으면서 각 글자를 정확히 한 개의 다른 위치에 있는 같은 글자와 짝지을 수 있다면 좋은 단어.

> [!success] How?
> 문자열을 순회하며 스택에 쌓음
> 스택의 마지막을 확인
>
> 1. 비어있다. -> 추가
> 2. 같다. -> 제거
> 3. 다르다. -> 추가
>    스택의 길이가 0이면 카운트
>    1이면 X
