> [!question] Question
> 수빈이의 위치가 X일 때, 갈 수 있는 위치
> $X-1, X+1, X*2$
> 수빈이가 동생을 찾을 수 있는 가장 빠른 시간이 몇 초 후인지

> [!success] How?
> 각 위치에 몇 초 후 도착하는지 배열로 나타내면 좋을듯
> 길이가 100,001인 배열을 만들자.
> 디폴트 값은 -1
> n번째 인덱스를 0으로 수정
> 큐에 n을 넣고 while문 수행
> for 문에 -1, +1, \*2를 넣고 수행

> [!info] Think
> 시간 배열을 만들고 각 연산을 수행하며 얼마나의 시간이 걸리는지 계산

> [!tip] Time Complexity
